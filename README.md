Cybersecurity Awareness Chatbot

A console-based chatbot designed to educate users about cybersecurity best practices through interactive conversations.

Features

- Voice Greeting: Plays a welcome audio message when the chatbot starts
- ASCII Art Logo: Displays a logo converted from an image file
- Typing Animation: Simulates realistic typing for bot responses
- Colored Console Output: Enhanced user interface with color-coded messages
- Interactive Q&A: Responds to user questions about cybersecurity topics

Prerequisites

- Windows operating system (for audio playback support)
- .NET Framework or .NET Core
- Audio file: `greeting.wav`
- Image file: `chatbotlogo.jpeg`

Installation

1. Clone or download this repository
2. Ensure the following files are in the project root directory:
   - `greeting.wav` - Welcome audio file
   - `chatbotlogo.jpeg` - Logo image for ASCII conversion

Usage

1. Run the application from Visual Studio:


2. Enter your name when prompted

3. Start chatting with the bot! Ask questions about:
- Passwords
- Phishing
- Safe browsing
- The bot's purpose

4. Type `exit` to end the conversation


Topics Covered

- Password Security: Creating and managing strong passwords
- Phishing Awareness: Recognizing and avoiding phishing attempts
- Safe Browsing: Identifying secure websites and safe browsing practices
